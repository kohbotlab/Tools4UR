# Tools4UR

A collection of tools and utilities for Universal Robots (UR).

Universal Robots（UR）向けのツールおよびユーティリティ集です。

---

### Overview
This repository provides a set of tools to support URScript development, deployment, and testing for Universal Robots. It is designed to be extensible, allowing for the addition of various UR-related utilities.

### Folder Structure
- **[program/](program/)**: URScript source files (`.script`). Contains example programs like palletizing logic.
- **[Script_Test/](Script_Test/)**: Python-based automation tools for sending URScript to URSim or real robots and performing automated testing with Modbus communication.

---

### 概要
Universal RobotsのURScript開発、デプロイ、およびテストをサポートするためのツール群です。今後、さまざまなUR関連ツールを追加しやすい構成になっています。

### フォルダ構成
- **[program/](program/)**: URScript（`.script`）のソースファイルを格納します。パレタイジングなどのサンプルプログラムが含まれています。
- **[Script_Test/](Script_Test/)**: URSimや実機ロボットへスクリプトを送信し、Modbus通信を利用した自動テストを行うためのPythonツール群です。
